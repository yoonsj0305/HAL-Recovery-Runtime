# Changelog

## 1.0.0 - Public PoC

- Froze the simulation-only CLI, artifact, safety, and exit-code contracts.
- Added a deterministic synthetic public example, release contract, internal-function PoC validator, reproducibility workflow, security guidance, threat model, artifact contract, and limitations documentation.
- Preserved explicit human review, dry-run-only promotion, artifact lineage, and the no-hardware boundary.

This is a public proof of concept, not a production runtime, hardware controller, real recovery system, or certification product.
